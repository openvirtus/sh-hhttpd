## HHTTPD

    Usage: hhttpd [OPTIONS...] [-b] DIRECTORY [PORT]
    
    This script helps launching different HTTP servers. This is usefull to
    test web apps.
    
       -v            : Show configuation.
       -s php        : Use PHP to service the directory.
       -s artisan    : Use artisan to serve Laravel website.
       -s mini_httpd : Use `mini_httpd` to serve site.
       -s busybox    : Use `busybox` to serve the site.
       -i composer   : Install composer, required to build PHP sites.
       -b            : Build (composer: composer.json).
    
       -d DOMAIN     : Domain to serve.
       -S            : Enable TLS/SSL.
    
    Examples:
    
       > httpd -s mini_httpd -d example.com -S . 441
